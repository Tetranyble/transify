

# Transify.io banking application

##How to setup

- clone the this repository and setup your database.
- run `composer install`
- next run `php artisan migrate --seed`
- start the dev server `php artisan serve`
- the application should be accessible at `http://localhost:8000`

- the default username: `e.lonard@transify.io` and password: `password`

However you could create your account from the signup screen.
